## 📋 О проекте

Официальный сайт-каталог дистрибьютора кухонной техники **ГЕФЕСТ** в Узбекистане. Сайт позволяет клиентам просматривать все модели техники, узнавать цены в сумах и долларах, и связываться с менеджером для заказа.

---

## ✨ Возможности

- 🛒 **Каталог товаров** — более 77 моделей техники
- 🔍 **Поиск по модели** — быстрый поиск внутри каждой категории
- 💱 **Двойные цены** — отображение цен в USD и UZS (сум)
- 📱 **Адаптивный дизайн** — оптимизирован для мобильных устройств
- 🏷 **Баннеры категорий** — красивые баннеры для каждого раздела
- 📞 **Быстрый контакт** — кнопки звонка и Telegram прямо в карточке товара

## 🚀 Технологии

- **HTML5** — структура сайта
- **CSS3** — стилизация и анимации
- **Vanilla JavaScript** — логика каталога, поиск, модальные окна
- **Inter** — основной шрифт (Google Fonts)
- **Vercel** — хостинг и деплой

## 🔄 Как обновить сайт

1. Внеси изменения в `index.html`
2. Загрузи файл в этот репозиторий
3. Vercel автоматически обновит сайт за ~30 секунд
